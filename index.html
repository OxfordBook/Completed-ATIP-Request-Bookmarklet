📋 Completed ATIP Request Bookmarklet Generator

Fill in your details below, then generate a custom bookmarklet to auto-fill the ATIP Informal Request form.

🔒 Privacy: Everything runs locally in your browser. No personal information is sent to any server — the generated bookmarklet embeds your data directly into the code you save as a bookmark. If you're still unsure, you can even disconnect from the internet while you use this page.
⚠️ Note: This bookmarklet works with the current ATIP form structure as of July 2026. If the Government of Canada updates the form, the bookmarklet may need updates to match the new field IDs.

🚀 How to Use This Bookmarklet

  1. 1 First, fill in all your details below and click Generate Bookmarklet. Then drag the generated button to your bookmarks bar (or copy the code into a new bookmark).
  2. 2 Find a previously completed ATIP request that you would like a copy of on https://open.canada.ca/en/search/ati.
  3. 3 Once you're on an "Informal Request for ATI Records Previously Released" form page, click the ATIP Auto-Fill bookmark(let) you created. It will automatically fill in all your information and (if you chose auto-submit) submit the request — all in one click!

📖 How to Install Your Bookmarklet

Chrome

  1. Click the ★ star icon at the right end of the address bar (or press Ctrl+D / ⌘D).
  2. In the dialog, give it a name like ATIP Auto-Fill.
  3. Click Edit (or More … then Edit).
  4. In the URL field, delete whatever is there and paste the bookmarklet code from above.
  5. Click Save. The bookmark will appear on your bookmarks bar.

Tip: Make sure your bookmarks bar is visible (Ctrl+Shift+B / ⌘+Shift+B).

Firefox

  1. Right-click on the bookmarks bar and select Add Bookmark…
  2. Give it a name like ATIP Auto-Fill.
  3. In the Location (URL) field, paste the bookmarklet code from above.
  4. Click Add. The bookmark will appear on your bookmarks bar.

Tip: If you don't see the bookmarks bar, press Ctrl+Shift+B (⌘+Shift+B on Mac) or right-click the title bar and check Bookmarks Toolbar.

Edge

  1. Click the ★ star icon in the address bar (or press Ctrl+D / ⌘D).
  2. Give it a name like ATIP Auto-Fill.
  3. In the URL field, paste the bookmarklet code from above.
  4. Click Done. The bookmark will appear on your bookmarks bar.

Tip: Show your bookmarks bar with Ctrl+Shift+B (⌘+Shift+B on Mac).
