Building on the work of https://tomcardoso.github.io/piggyback/ and https://github.com/tomcardoso/piggyback I built my own Completed ATIP Request Bookmarklet Generator. 

This tool lets you fill out your personal information just once when requesting previously completed ATIP requests from the Government of Canada. Using this Bookmarklet, you can submit requests in just one click. 

Use on: https://open.canada.ca/en/search/ati

Try my bookmarklet generator here: https://oxfordbook.github.io/Completed-ATIP-Request-Bookmarklet/

* Supports Auto-Submit after a timed delay (less clicks!)
* Partial French-language page support
* Updated list of Countries
* Bookmarklet will only submit on an EN/FR ATIP submission page. No risk of leaking your personal information if you run the bookmarklet on another webpage by mistake (though that is unlikely: the other website would have to coincidentally have the same form fields)
* Runs entirely in your browser. Nothing is sent to any servers.
